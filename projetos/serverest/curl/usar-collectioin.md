# Collection

O Postman Collection JSON, é um arquivo `json` contendo as operações de Cadastrar, Listar, Filtrar, Consultar, Atualizar e Excluir usuários na API do ServeRest.

Você poderá importar diretamente no Postman usando File → Import → Raw Text.

✅ Como usar esta collection

1. Abra o Postman.
2. Clique em Import → Raw Text.
3. Cole o JSON acima e clique em Import.
